# AI-College-Chatbot
AI College Chatbot is a Python-based chatbot that provides instant answers to common college-related questions using Natural Language Processing (NLP). It helps students access information about admissions, courses, fees, departments, faculty, placements, scholarships, and campus facilities through an interactive interface.
# Technologies Used
1. Python
2. NLTK
3. Database(mySQL)
4. Tkinter(GUI)
5. JSON (Knowledge Base)
